# 전문적인 Amazon A+ 콘텐츠 디자인 가이드

## 핵심 통계 및 성과
- Basic A+ 콘텐츠: 평균 3-10% 판매 증가
- Premium A+ 콘텐츠: 최대 20% 판매 증가  
- 고품질 A+ 콘텐츠는 반품률 감소 효과도 있음
- Google에서도 인덱싱되어 외부 트래픽 유입

## A+ 콘텐츠 이미지 사양 및 가이드라인

### 기본 이미지 규격
- **최소 해상도**: 970 x 600 픽셀
- **권장 해상도**: 1464 x 600 픽셀 
- **최대 파일 크기**: 5MB
- **파일 형식**: JPG, PNG
- **색상 모드**: RGB
- **DPI**: 72-300 DPI

### 모듈별 상세 규격

#### 1. Standard Image & Text Module
- **이미지 크기**: 970 x 600px (최소) ~ 1464 x 600px (권장)
- **텍스트 영역**: 이미지 옆 50% 공간
- **권장 텍스트**: 100-300단어

#### 2. Standard Four Image Text Module  
- **개별 이미지 크기**: 220 x 220px (정사각형)
- **총 4개 이미지**: 가로 배열
- **각 이미지별 텍스트**: 50-100단어

#### 3. Standard Single Image Module
- **이미지 크기**: 1464 x 600px (전폭 배너)
- **용도**: 브랜드 헤더, 라이프스타일 이미지
- **텍스트 오버레이 가능**

#### 4. Standard Comparison Chart Module
- **표 형식**: 최대 4개 제품 비교
- **헤더 이미지**: 각 220 x 220px
- **비교 항목**: 최대 8-10개 특징

## 디자인 베스트 프랙티스

### 1. 이미지 품질 기준
- **해상도**: 선명하고 고해상도 이미지 사용
- **배경**: 깔끔한 흰색 또는 브랜드 컬러
- **제품 비율**: 이미지의 80% 이상 제품이 차지
- **그림자/반사**: 자연스러운 그림자 효과 추가

### 2. 텍스트 디자인 가이드라인
- **폰트**: 읽기 쉬운 Sans-serif 폰트
- **크기**: 최소 14px 이상
- **색상**: 높은 대비율 (검정/진한 회색 on 흰 배경)
- **줄간격**: 1.2-1.5배
- **정렬**: 왼쪽 정렬 권장

### 3. 레이아웃 원칙
- **여백**: 충분한 화이트 스페이스 확보
- **계층구조**: 제목 > 부제목 > 본문 순으로 시각적 위계
- **색상 일관성**: 브랜드 컬러 2-3개로 제한
- **시각적 흐름**: 왼쪽 상단 → 오른쪽 하단 순 배치

## 고전환 A+ 콘텐츠 구조

### 권장 모듈 순서 (총 5-7개 모듈)
1. **Brand Story Module** - 브랜드 소개
2. **Standard Single Image** - 메인 헤로 이미지  
3. **Standard Image & Text** - 핵심 베네핏 #1
4. **Standard Four Image Text** - 4가지 주요 특징
5. **Standard Comparison Chart** - 제품 비교표
6. **Standard Image & Text** - 핵심 베네핏 #2
7. **Standard Image & Text** - 사용법/보증정보

### 콘텐츠 전략
- **첫 모듈**: 브랜드 스토리로 신뢰도 구축
- **중간 모듈**: 제품 특징과 베네핏 강조
- **마지막 모듈**: 구매 결정 도움 (보증, A/S 등)
