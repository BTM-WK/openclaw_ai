# A+ 콘텐츠 제작 워크플로우 & 체크리스트

## 프로젝트 시작 전 준비사항

### 1. 필수 정보 수집 체크리스트
- [ ] **제품 정보**
  - [ ] 제품명 및 브랜드명
  - [ ] 주요 특징 및 베네핏 (5-7개)
  - [ ] 타겟 고객층 정의
  - [ ] 가격대 및 포지셔닝
  
- [ ] **경쟁사 분석**
  - [ ] 상위 3개 경쟁사 A+ 콘텐츠 분석
  - [ ] 차별화 포인트 파악
  - [ ] 시장 내 포지셔닝 전략
  
- [ ] **브랜드 자산**
  - [ ] 고해상도 로고 파일 (.PNG, 배경 투명)
  - [ ] 브랜드 컬러 코드 (HEX)
  - [ ] 브랜드 폰트 (가능한 경우)
  - [ ] 기존 브랜드 가이드라인

### 2. 이미지 리소스 준비
- [ ] **제품 사진**
  - [ ] 메인 제품 이미지 (흰배경, 고해상도)
  - [ ] 다양한 각도 제품 사진 (5-8장)
  - [ ] 라이프스타일 이미지 (사용 상황)
  - [ ] 디테일 컷 (재질, 기능 등)
  
- [ ] **추가 비주얼 자료**
  - [ ] 인포그래픽용 아이콘
  - [ ] 배경 이미지 또는 텍스처
  - [ ] 모델 또는 사용자 이미지

## 제작 단계별 워크플로우

### Phase 1: 기획 및 설계 (1-2일)

#### Day 1: 컨셉 기획
- [ ] **콘텐츠 전략 수립**
  - [ ] 타겟 고객 여정 매핑
  - [ ] 핵심 메시지 우선순위 결정
  - [ ] 모듈 구성 계획 (5-7개)
  
- [ ] **와이어프레임 작성**
  - [ ] 각 모듈별 레이아웃 스케치
  - [ ] 텍스트 계층 구조 설계
  - [ ] 시각적 흐름 계획

#### Day 2: 콘텐츠 작성
- [ ] **카피라이팅**
  - [ ] 제목 및 헤드라인 작성
  - [ ] 베네핏 중심 설명문 작성
  - [ ] CTA 문구 개발
  - [ ] SEO 키워드 자연스럽게 포함

### Phase 2: 디자인 제작 (2-3일)

#### Day 3-4: 메인 모듈 디자인
- [ ] **Brand Story Module**
  - [ ] 브랜드 로고 및 스토리 이미지
  - [ ] 간결한 브랜드 소개 텍스트
  - [ ] 일관된 브랜드 톤앤매너
  
- [ ] **Hero Image Module**  
  - [ ] 임팩트 있는 메인 이미지
  - [ ] 제품 핵심 가치 메시지
  - [ ] 시선을 끄는 디자인

#### Day 5: 상세 모듈 디자인
- [ ] **Feature Modules (3-4개)**
  - [ ] Standard Image & Text 모듈
  - [ ] Four Image Text 모듈
  - [ ] 각 특징별 전문적인 시각화
  
- [ ] **Comparison Chart**
  - [ ] 제품 라인업 비교표
  - [ ] 명확한 차별화 포인트
  - [ ] 구매 결정 도움 정보

### Phase 3: 품질 관리 및 최적화 (1일)

#### Day 6: 품질 검토
- [ ] **기술적 검증**
  - [ ] 이미지 해상도 확인 (최소 970x600px)
  - [ ] 파일 크기 확인 (5MB 이하)
  - [ ] 색상 모드 확인 (RGB)
  
- [ ] **콘텐츠 검증**
  - [ ] 맞춤법 및 문법 검사
  - [ ] 브랜드 일관성 확인
  - [ ] Amazon 정책 준수 확인
