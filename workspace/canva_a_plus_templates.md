# Canva A+ 콘텐츠 전문 템플릿 가이드

## Canva에서 A+ 콘텐츠 제작하기

### 1. 기본 설정
- **새 디자인**: 사용자 지정 크기 선택
- **권장 캔버스 크기**: 1464 x 600 픽셀
- **배경**: 흰색 또는 브랜드 컬러

### 2. 템플릿 1: Standard Image & Text Module

#### 캔버스 구성 (1464 x 600px)
```
┌─────────────┬─────────────────────────────────┐
│             │                                 │
│   제품 이미지   │         텍스트 영역                │
│  (732x600)  │        • 제목 (32px)            │
│             │        • 부제목 (24px)           │
│             │        • 설명문 (16px)           │
│             │        • 불릿 포인트 (14px)       │
└─────────────┴─────────────────────────────────┘
```

#### 디자인 요소
- **제품 이미지**: 왼쪽 50% (732 x 600px)
- **텍스트**: 오른쪽 50%, 좌우 여백 40px
- **폰트**: Montserrat 또는 Open Sans
- **색상**: #000000 (제목), #333333 (본문)

### 3. 템플릿 2: Four Image Text Module

#### 캔버스 구성 (1464 x 600px)
```
┌─────────┬─────────┬─────────┬─────────┐
│ 이미지 1  │ 이미지 2  │ 이미지 3  │ 이미지 4  │
│(330x330)│(330x330)│(330x330)│(330x330)│
├─────────┼─────────┼─────────┼─────────┤
│  텍스트   │  텍스트   │  텍스트   │  텍스트   │
│  영역    │  영역    │  영역    │  영역    │
└─────────┴─────────┴─────────┴─────────┘
```

#### 디자인 요소
- **이미지 크기**: 각 330 x 330px
- **간격**: 이미지 간 27px
- **텍스트 영역**: 각 이미지 하단 200px
- **텍스트 정렬**: 가운데 정렬

### 4. 템플릿 3: 비교 차트 모듈

#### 캔버스 구성 (1464 x 600px)
```
┌────┬─────────┬─────────┬─────────┬─────────┐
│    │ 제품 A   │ 제품 B   │ 제품 C   │ 제품 D   │

│특징3│ 10시간  │ 15시간   │ 20시간   │ 25시간  │
└────┴─────────┴─────────┴─────────┴─────────┘
```

### 5. Canva 제작 단계별 가이드

#### 단계 1: 새 디자인 시작
1. Canva 접속 → "사용자 지정 크기" 클릭
2. 1464 x 600 입력 → "새 디자인 만들기"
3. 배경색 설정: 흰색 (#FFFFFF)

#### 단계 2: 제품 이미지 삽입
1. "업로드" → 제품 이미지 업로드
2. 이미지 크기 조정: 732 x 600px
3. 위치: 왼쪽 정렬

#### 단계 3: 텍스트 영역 생성
1. "텍스트" → "제목 추가"
2. 폰트: Montserrat Bold, 32px
3. 색상: #000000
4. 위치: 오른쪽 영역 상단

#### 단계 4: 브랜딩 요소 추가
1. 브랜드 로고 삽입 (우측 하단)
2. 브랜드 컬러 적용
3. 일관된 폰트 체계 유지

### 6. 전문적인 디자인 팁

#### 색상 팔레트 제안
```css
/* 미니멀 브랜드 */
Primary: #000000
Secondary: #FFFFFF  
Accent: #FF6B35

/* 프리미엄 브랜드 */
Primary: #2C3E50
Secondary: #ECF0F1
Accent: #E74C3C

/* 테크 브랜드 */
Primary: #34495E
Secondary: #3498DB
Accent: #1ABC9C
```

#### 타이포그래피 가이드
- **제목**: Montserrat Bold, 28-36px
- **부제목**: Montserrat Medium, 20-24px  
- **본문**: Open Sans Regular, 14-16px
- **캡션**: Open Sans Light, 12-14px

### 7. 산업별 전문 템플릿

#### 전자제품 템플릿
- 깔끔한 흰색 배경
- 제품 스펙 하이라이트
- 기술적 아이콘 사용
- 비교표 중심 구성

#### 홈&가든 템플릿  
- 따뜻한 색조 배경
- 라이프스타일 이미지
- 자연스러운 폰트
- Before/After 구성

#### 패션 템플릿
- 그라디언트 배경
- 모델 착용 이미지
- 세련된 타이포그래피
- 색상 바리에이션 표시

### 8. Canva Elements 활용

#### 아이콘 라이브러리
- "요소" → "아이콘" 검색
- 추천 키워드: "quality", "premium", "checkmark"
- 브랜드 컬러로 색상 변경

#### 도형 및 라인
- 구분선: 얇은 회색 라인 (1-2px)
- 강조 박스: 라운드 코너 사각형
- 배경 요소: 큰 원형 또는 기하학적 패턴

### 9. 모바일 최적화 체크

#### 반응형 디자인 확인
- 텍스트 크기: 최소 14px 이상
- 터치 타겟: 최소 44px 이상
- 이미지 품질: 고해상도 유지
- 로딩 속도: 파일 크기 5MB 이하

### 10. 품질 체크 및 내보내기

#### 최종 검토 사항
- [ ] 맞춤법 및 문법 확인
- [ ] 브랜드 가이드라인 준수
- [ ] 이미지 해상도 확인
- [ ] Amazon 정책 준수
- [ ] 전체적인 시각적 일관성

1. "다운로드" → "JPG" 선택
2. 품질: "고품질" 설정
3. 파일명: 모듈명_제품명_v1.jpg
4. 압축: 5MB 이하 확인
